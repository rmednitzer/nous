# Conformance posture: STANAG 4774 / 4778

**Adapter:** `src/nous/interop/stanag_4774.py` (BL-034)

**Standard:** STANAG 4774 (Confidentiality Label) and STANAG 4778
(Binding Label to Data). The spec lives in the NATO publication.

**v0.1 posture:** Encode wraps a payload in a JSON envelope with
`policyIdentifier`, `classification`, and `payload`. Decode parses
that JSON back into a dictionary.

**What is supported:** Label/payload coupling in JSON, suitable for
internal smoke tests. Default policy is `NATO`, default classification
is `UNCLASSIFIED`.

**What is omitted in v0.1:** XML rendering, ASN.1 binding, signing of
the label, the full classification ontology (categories, releasability
markings, caveats).

**Conformance claim:** None. v0.1 is a shape-only adapter; real
STANAG conformance requires a certified implementation.
